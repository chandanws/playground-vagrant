## Prerequisites

* Vagrant
  * Packages can be dowloaded [here](https://www.vagrantup.com/downloads.html).

## Creation

`cd` to the project folder and

    vagrant up
    
## Using the VM

If needed, access with `vagrant/vagrant`.

`startx` to launch the graphical environment.
